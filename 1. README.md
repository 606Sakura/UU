# Probability Simulation Toolkit

一个用 Python 实现的概率模拟工具包，包含经典概率问题模拟，适合教学、学习和实验。

## 功能
- 蒙特卡洛方法估算 π
- 生日悖论概率模拟
- 多骰子点数和分布模拟
- 通用抽奖/概率事件模拟（支持保底机制）
- 结果可视化（matplotlib）

## 安装
```bash
git clone https://github.com/yourusername/probability-simulation-toolkit.git
cd probability-simulation-toolkit
pip install -r requirements.txt
# 或可安装为包
pip install .
from probability_sim.pi_estimator import MonteCarloPiEstimator

sim = MonteCarloPiEstimator(points=100000)
pi, fig = sim.run()
print(f"估算 π: {pi:.6f}")
fig.show()
python examples/run_all.py
